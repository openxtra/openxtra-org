Welcome to Free Stuff.

Projects that OPENXTRA has created over the last few years. Ranging from a small Wordpress widget for displaying the most popular articles on a blog, all the way up to a full feature time series database. If you'd like to know more about any of the projects, please get in touch.

LogFilter

Logfilter is a simple command line log file filter tool for Apache web server. The project is now dead though the source code is still hosted at GitHub for those who are interested. Please feel free to fork.

OPENXTRA Basics

OPENXTRA Basics was a free software product produced by OPENXTRA Limited all the way back in 2003. It comprised a number of open source tools of use to the systems administrator or network manager.

The package included network mapping, network monitoring and packet capture tools. It was all packaged together in a single installer with a cute Windows based GUI called OPENXTRA Commander.

OPENXTRA Console

OPENXTRA Console is a convenient way to quickly and easily view the readings from the Sensatronics range of temperature & environmental monitoring equipment using Google Desktop.

TimeTag Powershell time series database

TimeTag is an open source time series database implemented from the ground up for Windows PowerShell. If you want to know more about TimeTag, please visit the dedicated TimeTag project website.

OPENXTRA HotSpot Sidebar Gadget

OPENXTRA HotSpot Sidebar Gadget is a convenient way to quickly and easily view the readings from the Sensatronics range of temperature & environmental monitoring equipment.

Windump-XTRA

Windump-XTRA was an easy to install distribution of WinDump. It has now been discontinued.

Net-SNMP-XTRA

Net-SNMP-XTRA was an easy to install distribution of Net-SNMP. It has now been discontinued.

Ethereal-XTRA

Ethereal-XTRA was an easy to install distribution of the Ethereal project. It has now been discontinued.

NTop-XTRA

NTop-XTRA was an easy to install distribution of NTop for Microsoft Windows. It has now been discontinued.

MRTG-XTRA

MRTG-XTRA was an easy to install package of the MRTG project for Microsoft Windows. It has now been discontinued.

Popularity Contest Widget

Popularity Contest is a WordPress widget for displaying the most popular posts on a blog sidebar. The project is hosted on the Wordpress development site.
